Selected Software Development Portfolio

ApplyAid – Academic Application Support Platform

Designed and developed ApplyAid, a full-stack Django-based platform with PostgreSQL integration and RESTful APIs, enabling efficient management of academic application data and user-driven task automation.

I developed ApplyAid, a full-stack web platform aimed at helping users organize and manage the academic application process. The platform allows users to search for universities and professors, build and store their personalized database, and set automated reminders for key deadlines and tasks.

I built the back-end using Python and the Django framework, with a PostgreSQL database to ensure structured, reliable data storage. To enable seamless client-server communication, I implemented a RESTful API using Django REST Framework. On the front-end, I used HTML, CSS, and JavaScript to create a clean, responsive user interface.

I also managed the entire development workflow with Git and GitHub. This project highlights my ability to design and deliver practical, scalable solutions with a strong emphasis on back-end architecture, database management, and API development.

Navaro – E-Commerce Platform for Digital Devices

Designed and developed Navaro, a Django-based e-commerce platform backed by a MySQL database, featuring custom data models and a responsive user interface guided by Figma prototyping.

I developed Navaro, an e-commerce web application tailored for browsing and purchasing digital devices. I designed the back-end using Python and the Django framework, integrating a MySQL database for managing products, users, and transactions with high reliability and consistency.

The front-end was built using HTML, CSS, and JavaScript, providing users with a smooth and interactive shopping experience. I also utilized Figma during the UI planning phase to prototype and refine the visual layout before implementation.

This project showcases my ability to create structured, database-driven systems and deliver user-centered features, emphasizing strong back-end logic and practical front-end integration.
